# 웹서버와 웹애플리케이션 서버

### 프록시
클라이언트와 서버 사이에서 중계자 역할을 하며 대리로 통신하는 것
- **프록시 서버**
  - 중계 역할을 하는 서버(컴퓨터)
- **포워드 프록시**
  - 클라이언트 편, 서버와의 통신 중계
  - ip 우회, 클라이언트의 익명성 보장
  - 콘텐츠 제한
- **리버스 프록시**
  - 서버 편, 클라이언트와의 통신 중계
  - 로드밸런싱(부하 분산)
  - 원본 서버의 ip를 숨김, 공격으로부터 보호
  - 캐싱

### Cloud Computing
- 인터넷을 통해 데이터 저장, 처리, 그리고 응용 프로그램을 사용할 수 있게 하는 기술
- 로컬 컴퓨터에 소프트웨어나 데이터를 설치하거나 저장할 필요 없이, 원격 서버에서 필요한 자원과 서비스를 제공받을 수 있음

# AWS

## AWS(Amazon Web Services)
컴퓨팅, 스토리지, 데이터베이스와 같은 인프라 기술부터 기계 학습 및 인공지능, 데이터 레이크 및 분석, 사물 인터넷을 IT서비스로 제공

가입 완!!

## Amazon EC2 (Elastic Compute Cloud)
- 컴퓨팅 파워의 규모를 자유자재로 변경할 수 있는 컴퓨팅 서비스
- 새로운 서버 인스턴스 확보 및 부팅 시간을 단축
- Virtual Machine
- Linux 또는 Windows, Ubuntu 등 다양한 OS 지원
- 사용하는 용량만큼 비용 지불

### Amazon EC2 인스턴스
- AWS 클라우드 환경의 가상 서버
- 애플리케이션을 호스팅하거나 다양한 작업을 수행
- 필요에 따라 생성, 중지, 종료

## AMI (Amazon Machine Image)
- Amazon EC2 인스턴스를 설정하고 부팅하는 데 필요한 소프트웨어를 제공하는 이미지
- 인스턴스를 시작할 때 AMI를 지정
- 동일한 구성의 여러 인스턴스가 필요할 때 단일 AMI에서 여러 인스턴스를 시작
